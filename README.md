🎬 Anime Checklist App

A full-stack web application to manage your anime watchlist — built as a learning project to practice modern web development.

⸻

🚀 Features
	•	🔐 User Authentication (Signup / Login using JWT)
	•	➕ Add anime to your list
	•	✏️ Edit anime details
	•	❌ Delete anime
	•	⭐ Mark as favourite
	•	📊 Dashboard with stats (total, completed, favourites, ongoing)
	•	👤 User profile page
	•	🔍 Basic filtering (watch status, favourites)

⸻

🛠 Tech Stack

Frontend
	•	Next.js (App Router)
	•	React
	•	Tailwind CSS

Backend
	•	Node.js
	•	Express.js
	•	MongoDB (Mongoose)
	•	JWT Authentication
	•	bcrypt (password hashing)
